🏫 School Management API

Welcome! Here are all the available endpoints:

1. Add School

POST /addSchool

Request Body (JSON):

{
  "name"      : "string  (required) — Name of the school",
  "address"   : "string  (required) — Address of the school",
  "latitude"  : "number  (required) — Latitude of the school",
  "longitude" : "number  (required) — Longitude of the school"
}

Example Request:

{
  "name"      : "ABC School",
  "address"   : "Indore, Madhya Pradesh",
  "latitude"  : 22.7196,
  "longitude" : 75.8577
}

Example Response:

{ "message": "School added successfully" }

2. List Schools

GET /listSchools

Query Parameters:

latitude  = number (required) — Your current latitude
longitude = number (required) — Your current longitude

Example Request:

/listSchools?latitude=30.7196&longitude=45.8577

Example Response:

[
  {
    "id"        : 1,
    "name"      : "ABC School",
    "address"   : "New York",
    "latitude"  : 40.7128,
    "longitude" : 74.0060,
    "distance"  : 0.0
  }
]
